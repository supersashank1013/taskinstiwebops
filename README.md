# Lumière — Product Store

A luxury-styled product catalogue built with Vite + React, fetching data from the DummyJSON Products API.

## Features

- Fetches 100 products from `https://dummyjson.com/products`
- Real-time search by title, description, or brand
- Category filter pills (populated dynamically)
- Sort by: Featured / Price / Rating / Discount
- Shimmer skeleton loading state
- Graceful image error fallback
- Stock status badges (In Stock / Low Stock / Sold Out)
- Discount badges & crossed-out original prices
- Animated card entrance with staggered delays
- Smooth hover lift effects
- Fully responsive grid (4 → 3 → 2 → 1 columns)
- CSS Modules for scoped styling

## Tech Stack

- **Vite** 5 + **React** 18
- CSS Modules
- Google Fonts: Cormorant Garamond + DM Sans

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
